# database_sql
Baza de date relizata in acezst proiect are ca scop gestionarea datelor unei firme care realizeaza evenimente. Aceasta permite administrarea informatiilor legate de evenimente, de participanti, de angajatii care se ocupa de organizare si de clienti.

•	EVENIMENT – stocheaza informatii despre evenimentele organizate (denumire, durata, data de începere).
•	LOCATIE – contine date despre locatiile disponibile pentru desfasurarea evenimentelor.
•	CLIENT – gestioneaza datele clientilor care participa la evenimente.
•	ANGAJAT – contine informatii despre angajatii companiei.
•	EVENIMENT_CLIENT – tabel de legatura pentru relatia many-to-many dintre eveniment si client.
•	EVENIMENT_ANGAJAT – tabel de legatura pentru relatia many-to-many dintre eveniment si angajat.
•	FURNIZOR- stocheaza informatii despre firmele ce furnizeaza alimemnte/ echipamemnte/ materiale pentru realizarea sau organizarea evenimemntului
